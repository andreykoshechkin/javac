
Exception (Исключение) - обработка сценария ошибки.

Checked Exception (Complite time Exception ) - Исключение, которые возникают при работе с внешними источниками,
так как программа не имеет возможности контролировать то, что находится за ее пределами(БД, Сети, какие-то файлы) ТРЕБУЕТ ОБЯЗАТЕЛЬНОЙ ОБРАБОТКИ

Unchecked Exception  (Runtime Exception ) - Исключение, которые не обязательно обрабатывать так как, runtime exception возникают из-за ошибкой в коде.
возникают во время выполнение программы (к примеру, деление на нуль)

StackTrace - описывает от куда пришел exception
