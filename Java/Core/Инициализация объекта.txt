Наследование			Потомок всегда вызывает конструктор родителя, даже если конструктор родтеля без параметров, просто это проиходит не явно.
				Если конструктор родителя имеет параметры то потомок явно должен его вызвать через оператор super()


Инициализация объекта 		Вызов конструктора происходит вверх по иерархии,а инициализация происходит вниз по иерархии. 
				То есть, когда мы инициализируем объект, то этот объект вызывает конструктор родителя, 
				в свою очередь родителя вызывает конструктор класс Object, потом происходит инициализация конструктора класса Object,
				после чего проинициализируется конструктор родителя, и только после этого проинициализируется конструктор потомка
