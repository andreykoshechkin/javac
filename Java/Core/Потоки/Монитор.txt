Монитор в Java - это некая сущность, которая блокирует доступ к определенному ресурсу, нескольким потокам одновременно

Как работает монитор - при входу в синхронизированный блок кода, поток захватывает монитор, если монитор занят, то поток будет ждать, когда монитор освободится
Как только поток завершает выполнение синхронизированного блока кода, монитор автоматически освобождается.

Монитор позволяет избежать состояние гонки.
