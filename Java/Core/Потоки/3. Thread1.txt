Thread - класс многопоточности! 
методы:
run() - позволяет задать какую-то логику потока
start() - позволяет создать поток и запустить метод run
 у Thread есть метод public void run() который нужно переопределить
Потоки нужны для того, выполнять  какие-то действия одновременно 
потоки не синхронизированны, по этой причине, два потока могут пересикаться друг с другом

Sleep() - останавливает поток на какое-то время 
join() - данный метод указывает главному потоку, что нужно дождаться завершения потока
