CountDownLatch - средство синхронизации, которое является неким замком с обратным отсчетом, CountDownLatch действует таким образом, что один набор потоков, будет ждать
пока второй наборо потоков не завершит свои действия. Имеет счетчик, который уменьшается

CountDown - метод котоырй позволяет уменьшать счетчик
await - ждет пока счетчик не достигнет нуля(под счетчиком имеется ввиду тот аргумент который мы написали при создание объекта CountDownLatch )
