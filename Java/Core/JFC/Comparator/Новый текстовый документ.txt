Comparable:			
 — Это функциональный интерфейс с одним единственным методом compareTo, который возвращает int. Реализация comparable проиходит на уровне класса.		

Возвращает int потому-что:
 — Если объекты равны то вернет 0, 
 — Если объекты больше то вернет 1
 — Если объекты меньше вернет -1, 
 


Comparable - реализуется внутри класса. По сути, определяет обычный/естественный порядок сравнения объектов. Сортировка происходит в одном единственном варианте.

Comparator -  Можно реализовать различные варианты сортировки, основанные на сравнении различных полей.

Comparator позволяет так же задать сортировку по нескольким значениям.
   List<Person> list = new ArrayList<>();
        list.add(new Person(1, "Ivanaaa"));
        list.add(new Person(3, "Lena"));
        list.add(new Person(2, "Sveta"));

        list.sort(Comparator.comparing(Person::getName).thenComparing(Person::getId));	<-- Сначала сортировка по name потом по id
