Set - это множество, которое может иметь только уникальные элементы. В основе Set лежит упрощенная версия map, так как Set не использует пару ключ/значение но при этом, когда 
мы добавляем данные в set они хранятся ввиде ключа и заглушка ввиде константы.
Set позволяет реализовать математическое множество, т.е мы можем объеденить несколько множеств в одно, получить пересечени или получить разность множеств.



HashSet Хранит в себе HashMap, и при добавление элемента, HashSet будет выглядеть как Ключ и заглушка ввиде константы.
В качестве ключа может быть Null так как Set внутри себе содержит Map, а в интерфейсе Map ключь может быть null

Методы: Реализация происходит за константное время.

contains() - возвращает true если элемент есть в нашем сете. 

addAll - объединяют множества.
retainAll - находит тот элемент, который пересекается.
removeAll - находит разность множеств(Отнимаем set1 - set2);
