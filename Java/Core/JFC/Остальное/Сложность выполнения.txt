ArrayList:
— Быстрый доступ к элементам по индексу за время O(1);
— Доступ к элементам по значению за линейное время O(n);
— Медленный, когда вставляются и удаляются элементы из «середины» списка;
— Позволяет хранить любые значения в том числе и null;

Хеш-таблицей называется структура данных, реализующая интерфейс ассоциативного массива (абстрактная модель «ключ – значение» или entry)
Node — это вложенный класс внутри HashMap, который имеет следующие поля:
поиск по значение O(n)


HashMap:
— Удаление, добавление и поиск элементов осуществляется за константное время
