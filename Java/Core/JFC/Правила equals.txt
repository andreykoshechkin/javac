Симметричность:
Ссылочная переменные a и b вернут true только тогда, когда b и a вернет true    

Рефлексивность: 
объект всегда равен самому себе 

Транзитивность:
Если у нас есть 3 ссылочные переменные a b c то правило гласит следующее: 
Если a b возвращает true и b c возвращает true, тогда a c должны вернуть true


Постоянство:
Объекты которые мы сравниваем либо возвращают true либо возвращают false.
