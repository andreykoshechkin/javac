Синхронизированный коллекции - это те коллекции, которые имеею безопасные методы для использования в многопоточной среде

HashTable - устаревший синхронизированный класс, в отличие от HashMap, пара ключ значение не могут быть null. Внутри себя, так же как и map, хранит массив из связных списоков
В каждой ячейке массива, лежит связной список.

Stack(Класс) - это синхронизованный класс, который представлен ввиде обратной очереди.Это класс который следует правилу Последний зашел первый вышел.
