Buffer  -  это блок памяти в которую мы может записывать информацию и читать информацию.
Channel - некий мост, который передает информацию.

Buffer имеет два режима			//Режим чтения и режим записи. Position и Limit зависит от того в каком режиме находится Buffer



Записью.		//Channel Читает данные из файла и записывает в Buffer
Чтение.			//Channel читает информацию из Buffer и записывает ее в файл.

Отличие NIO от IO 	в том, что IO работает в одном направление(не может одновременно и читать и записывать), а Channel and Buffer в оба.



Position 		//Для Buffer это место от куда мы начинается запись в буффер, после метода flip, position обновляется и начинается с нуля
			  и далее Channel считывает байты с нулевой позиции.

Limit 			//Максимально количество байтов которые мы можем записать в буффер. К примеру у нас осталось 3 байта, то limit переместиться на 3 позицию, и в режиме
			  чтения он автоматически будет на 3 позиции.

Метод:			//flip() меняет режим с записи на чтение, при этом, чтение начнется с нуля.
			//Clear() перезаписывает старые элементы на новые 
			//hasRemaining() пока у буффера есть что читать - возвращаеи bool

    RandomAccessFile randomAccessFile = new RandomAccessFile("C:\\Users\\79964\\Desktop\\text.txt", "rw");
        FileChannel channel = randomAccessFile.getChannel();
        ByteBuffer buffer = ByteBuffer.allocate(20); -------- Устанавливаем размер, какое количество байт мы можем разм считать.
        int x = channel.read(buffer); ------ записывам количество символов из файла.
        StringBuffer StringBuffer = new StringBuffer();

        while (x > 0) {
            buffer.flip();	---- переворачиваем(чтение начнется с 0ого байта) и переходим в режим чтения.
            while (buffer.hasRemaining()) { ----- проверям, есть ли какие-то байты дальше.
                StringBuffer.append((char) buffer.get()); ---- записываем и преобразуем символы 
            }
            buffer.clear();		---- на старые записи накладывем новые записи
            x = channel.read(buffer);;	---- перезаписываем количество символов из файла.
        }
        System.out.println(StringBuffer);
