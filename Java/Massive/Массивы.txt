Одномерный Массив:(Массив в котором мы можем сразу обращаться к элементам по его индексу)
int [] totalNum = {5,2,3};

int sum = 0;
for (int sumNum : totalNum){
sum = sum+sumNum;

}
System.out.println(sum);
