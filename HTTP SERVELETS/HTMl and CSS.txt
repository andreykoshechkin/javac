<h1> Загаловок<h1>		<-Устанавливаем заглавный текст
 <p>какой-то текст<p/>	<-Внутри тега пишем текст
<i><Курсив<i>		<- сделает текст/слово Курсивным
<em><Курсив<em>		<- сделает текст/слово Курсивным
<br>			<-переноса строки
<hr>			<-горизонтальной линии 
<div> Анкета </div>		<-Устанавливаем заглавный текст
<ul> </ul>			<--Представление данных ввиде списка
<li> </li>			<--Значение списка
 			
<p></p> - парный, определяет текстовый абзац (от слова paragraph)
<b></b> - парный, устанавливает жирное начертание шрифта (от слова bold)
<i></i> - парный, устанавливает курсивное начертание шрифта (от слова italic)
<br> - одинарный, устанавливает перевод строки в том месте, где этот тег находится (от слова break)
<title>Страница</title> - на вкладке появится название Страница


Пример:
<h1> HTML TEXT<h1>
  <p>Строк 1<i> курсив <i>вторая строка <i>курсив<i/><p/>
