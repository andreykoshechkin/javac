JSON Создержит пару ключ, значение. Ключ и значение разделяются двоеточием ":". Ключ является строкой
Сложные объекты, такие как request, start-row, headers, body и тд, разделются "{ }".
Ключ и значение пишется внутри фигурных скобок.
В отличие от xml где мы писали <request, в json достаточно просто открыть скобки { } и внутри этих скобок, описывать сложные объекты
Один сложный объект от другого разделяется запятой
  
Пример:
{
  "id": 25,
  "StartRow": {
    "method": "GET",
    "url": "www.google.com",
    "version": "1.1"
  },
  "headers": [
    {
      "name": "content-type",
      "value": "text/html"
    },
    {
      "name": "accept",
      "value": "application/json"
    }
  ],
  "body": {
    "type" : "text/plain"
  }
}

headers содержит в себе массив и разделяется []
