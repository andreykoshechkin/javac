Полное название ISO/OSI
ISO - международная организация по стандартизации компьютерных сетей
OSI - эталонная модель, которая дает сетевым устройствам возможность общаться друг с другом
Model osi - компьютер который передает данные идет вниз по модели osi, компьютер который принимает идет вверх по модели 

7. Прикладной урвоень: обеспечивает связь между приложеньями, так же, этот уровень взаимодействует с данными от пользователя.
   Протоколы:
   HTTP - Основной протокол для передачи информации
   FTP  - Для передачи файлов
   SMTP - Протокол для передачи почты
   SSH  - Для удаленного доступа


6. Уровень представления: Обеспечивает шифрование, дешифрование,  компрессию данных, декомпрессию.
   Протоколы:
   TLS - Протокол защиты транспортного уровня. Данные протокол основан на SSL
   SSL - Криптографический протокол

 
5. Уровень сеанса: Отвечает за открытие/закрытие соединений
   Протоколы: Протоколы сенсового уровня не выделяются, их работу выполняют протоколы вверхнего уровня


4. Транспортный уровень: разбивает потоки данных(массив байт) на сегменты и наооборот, склеивает пакеты из сетевого уровня в сегменты
    Протоколы:
    TCP - Один из основных протоколов передачи данных
    UDP - Небезопасный протокол для передачи данных


3.Сетевой уровень: разбивает сегменты на пакеты, а так же передает эти пакеты межды разных сетей, делается это с помощью маршрутизатора 
    Протоколы: 
    IP - главный протокол (Internet protokol)


2. Канальный уровень: преобразует пакеты, которые ему поступили от сетевого уровня на фреймы(Кадры) и далее передает эти фреймы по одной сети, это делается с помощью комутатора
   Протоколы:
   Ethernet:
   PPP:
   PPPoE: 

1. Физический уровень: преобразует фреймы в биты и отвечает за доставку по физическим каналам(кабель, радиоэфир)
