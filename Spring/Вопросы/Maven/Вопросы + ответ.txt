Maven - это программа для автоматической сборки проектов, которая состоит из савокупности plugin, а plugin содежат в себе голы. Голы это обычные java классы 
SuperPom - это главный pom.xml файл, в котором указано откуда скачивать зависимости и плагины
Project lifecycle - В Maven существует 3 жизненных цикла, это Сlean, Build, Site,  каждый из которых состоит из phase, а уже phase содержит в  себе goal
