PartThreeJpaQuery	Отвечает за создание запросов на основании имени метода, под копотом парсит название метода на основании регулярных вырожений, который по умолчанию определены.
NamedQuery		Класс, который отвечает за создание именованных запросов, для реализации нужнр над классом поставить аннотацию @NamedQuery и написать сам запрос (параметр name должен следовать шаблону).
NativeJpaQuery		Класс, который отвечает за нативные запросы, для реализации нужно поставить аннотацию @NativeQuery
