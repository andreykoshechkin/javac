# model.addAttribute:
  Добавляет атрибуты к модели для текущего запроса.
  Данные доступны только для текущего запроса и не сохраняются после перенаправления.
  Подходит для передачи данных в представление, которые должны быть доступны только в текущем запросе.


# redirectAttributes.addFlashAttribute:
  Добавляет атрибуты, которые будут доступны после выполнения редиректа.
  Атрибуты сохраняются во flash scope (временном хранилище), обеспечивая доступность только на следующем запросе после редиректа.
  Подходит для передачи данных между запросами, когда необходимо сохранить данные только на короткое время (обычно для передачи сообщений об успехе или ошибке).
