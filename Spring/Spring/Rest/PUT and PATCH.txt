Разница между PUT и PATCH
Основное различие между PUT и PATCH в REST API заключается в том, 
что PUT обрабатывает обновления, заменяя весь объект , 
в то время как PATCH обновляет только те поля, которые вы ему предоставляете .

Если вы используете метод PUT, вся сущность будет обновлена. В большинстве REST API это означает,
 что все отсутствующие поля будут перезаписаны пустыми значениями. С другой стороны, метод PATCH обновляет отдельные поля, не перезаписывая существующие поля.
