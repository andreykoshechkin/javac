Spring Framework предоставляет два основных интерфейса для управления бинами (компонентами) и контейнерами в приложениях: BeanFactory и ApplicationContext.

BeanFactory		# Базовый интерфес для управление бинами, является не совсем актуальным, так как предоставляет минимальный набор функциональности, 
                          вместо BeanFactory лучше использовать ApplicationContext

ApplicationContext	# Базовый интерфейс, который реализует BeanFactory. Этот интерфес дает больше возможностей для управление бинами
