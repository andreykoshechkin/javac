Конечно, вот основные команды для работы с Git:

git init: Инициализация нового репозитория.

git clone <URL>: Клонирование существующего репозитория на локальный компьютер.

git add <файлы>: Добавление файлов в индекс (staging area) для подготовки к коммиту.

git commit -m "Сообщение коммита": Фиксация изменений, добавленных в индекс, в локальном репозитории с сообщением.

git status: Просмотр текущего статуса репозитория, включая измененные, добавленные и неотслеживаемые файлы.

git pull: Получение обновлений с удаленного репозитория и объединение их с локальными изменениями (если они есть).

git push: Отправка локальных изменений в удаленный репозиторий.

git branch: Просмотр списка локальных веток и определение текущей активной ветки.

git checkout <ветка>: Переключение на указанную ветку.

git merge <ветка>: Объединение указанной ветки в текущую ветку.

git log: Просмотр истории коммитов.

git diff: Просмотр изменений между рабочей директорией и последним коммитом.

git remote -v: Просмотр списка удаленных репозиториев, связанных с вашим репозиторием.

git fetch: Получение обновлений с удаленных репозиториев без их объединения.

git reset <файл>: Снятие файлов с индекса.

git rm <файл>: Удаление файла и фиксация этого изменения.

git stash: Сохранение временных изменений в "stash", чтобы позже их применить.
