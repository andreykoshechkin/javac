Настройка SSH 	//Нужна для того, чтобы мы могли синхронизировать наш удаленный репозиторий с локальным и при этом, не вводить каждый раз свой пароль от уч.записи github.

Шаги для настройки SSH соединения:
1. Нужно проверить наличие существующих ключей. Открыть gitBash и ввести команду  ls -al ~/.ssh
2. В gitBash ввести команду ssh-keygen -t ed25519 -C "email, который используется на gitHub" 
3. В gitBash ввести ssh-add C:\Users\YOU/.ssh/id 					     


(прим. ssh-keygen -t ed25519 -C "koshechkinandrey9@gmail.com")
(прим. ssh-add C:\Users\YOU/.ssh/id_ed25519)
================

ssh-add -l 	    //Эта команда отобразит список добавленных ключей в агент SSH вместе с их идентификаторами. Если агент работает и содержит ключи, вы увидите их в этом списке.
eval $(ssh-agent)   //Эта команда запустить агента
ssh-add C:\Users\YOU/.ssh/id_ed25519	//Добавит ключ к агенту

clip < ~/.ssh/id_ed25519.pub		#Команда которая скопирует ключ. Этот ключ мы вставим на странице добавления ssh Ключа на Git
