Спецификация				# Это набор интерфесов и аннотаций(классов), которые мы должны реализовывать, чтобы соблюдать эту спецификацию 
					  Примерами спецификаций в мире Java являются Java EE спецификации, которые определяют стандарты для разработки серверных приложений, 
					  и Java Servlet API, который предоставляет спецификацию для создания веб-приложений на платформе Java.


JavaEE (Java Enterprise Edition)	# Набор спецификация, которая описывает архетектуру серверного приложение и является дополнением JavaSE 
					  Java EE включает в себя множество спецификаций, таких как Java Servlet, JavaServer Pages (JSP), 
					  Enterprise JavaBeans (EJB), Java Messaging Service (JMS), Java Persistence API (JPA), и другие.
					  Эти спецификации определяют стандарты и интерфейсы, которые разработчики могут использовать для создания приложений,
					  которые легко масштабируются и поддерживают высокую доступность.


JavaSE (Java Standart Edition)		# Java SE (Java Platform, Standard Edition) представляет собой базовую платформу языка программирования Java, 
					  предназначенную для создания приложений настольного уровня, встраиваемых систем,
					  мобильных устройств и других разнообразных приложений.
