Спиcок Listener лежат в классе FastSessionServices.

Есть два основных понятие:
 1. Event    - это некий объект с набором полей. В качестве Event может выступать любой объект, например наше DTO.
 2. Listener - слушатель событий, который слушает соответствующий Event, в каждом Listener есть свой один метод, в котором описывается логика.

Мы моженм создавать много Listener на один Event.
