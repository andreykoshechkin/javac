Аннотация @Builder.Default используется для указания значения по умолчанию для поля, 
которое генерируется при использовании паттерна "Builder". Это означает, что при создании объекта с помощью паттерна "Builder", 
если значение для поля не было установлено явно, то будет использоваться значение, указанное в @Builder.Default.


StudentEntity student = StudentEntity.builder()
    .firstName("John")
    .lastName("Doe")
    .birthdate(LocalDate.of(1990, 1, 1))
    // Не указываем toCoursesList, будет использован пустой список по умолчанию
    .build();
В этом примере toCoursesList будет инициализирован пустым списком, так как мы не установили его явно при создании объекта.
