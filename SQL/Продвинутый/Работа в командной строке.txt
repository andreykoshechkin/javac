1: пуск -> найти -> SQL Shell -> Открываем
2: Вводим данные
		==localhost
		==Имя нужной базы данных
		==Порт 
		==Пароль

Пример:
Server [localhost]: localhost --- подключение к хосту
Database [postgres]: flight_repository	--- подключение к конкретной базе
Port [5432]: 5432 --- порт
Username [postgres]: postgres --- имя
Пароль пользователя postgres: --- пароль

Команды:
 \l - Позволяет отобразить все базы данных на текущем хосте(на хосте localhost)
 \d - Показать все таблицы(в том числе и sequence)
 \dt - Показать только таблицы без sequence)
 \c имяДругойБазы - позволяет поменять одну базу данных на другую
