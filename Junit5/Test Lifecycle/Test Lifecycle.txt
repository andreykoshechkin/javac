BeforeEach 	//Устанавливается, когда  нам, перед каждым тестом нужно что-то установить. Выполняется в циксле перед каждым тестом.
AfterEach	//Устанавливается, когда  нам, посде каждого теста нужно что-то установить. Выполняется в циксле перед каждым тестом.
BeforeEach -> @Test -> AfterEach

Сколько тестов у нас будет, столько раз мы вызовем цикл BeforeEach/AfterEach

====================================================================================================================================
BeforeAll	//Устанавливается, когда  нам, перед всеми тестами нужно что-то установить. Выполняется один раз в самом начале.  Обязятаельно писать только в начале
AfterAll	//Устанавливается, когда  нам, после всех тестов нужно что-то установить. Выполняется один раз в самом конце.	  Обязятаельно писать только в конце

Пример: 
Устанавливаем в самом начале -> Что-то устанавливаем в начале для конкретного теста -> Выполняем тест -> Что-то устанавливаем в конец для конкретного теста ->
снова что-то устанавливаем в начале для конкретного теста  -> выполняем другой тест -> снова что-то устанавливаем в конец для конкретного теста -> Устанавливаем что-то в конец.
BeforeAll -> BeforeEach -> @Test -> AfterEach -> BeforeEach -> @Test -> AfterEach ->AfterAll
